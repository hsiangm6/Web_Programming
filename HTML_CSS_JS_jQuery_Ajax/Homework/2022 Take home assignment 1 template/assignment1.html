教育目標

本院願景 : 卓越研究、教學、社會與實務影響力成為全球認可之管理學院。

  1. 發展優質研究環境 : 促進並維持管院優質環境，鼓勵教師與國際學者合作。
  2. 鼓勵教學創新與多元化學習 : 透過卓越教學方法，使學生具備創新能力、批判性思考及自我反思。
  3. 推動社會關懷和發揮影響力 : 提升社會關懷和社會影響力，培養學生ESG的社會責任。
